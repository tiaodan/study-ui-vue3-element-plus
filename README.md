# 安装vue3+elementplus教程
1.下node，windows版本，双击安装。用加入环境变量？
2.下开发工具。vs code
3.下载谷歌浏览器，作为开发环境
4.导入vue
 - 新手入门，用cdn导入方式
- cdn练习成功后，用npm构建项目。cmd命令行，切换到项目目录,所在的上级目录（不要自己先创建项目目录，用npm创建），运行npm create vue@latest填完项目后，全部选回车选No

# 安装vue3
npm create vue3@latest
npm install
npm run dev
